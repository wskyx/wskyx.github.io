Hub usb prodigy.cat PA-01 review

Published on 3 November 2025 05:22 PM
#article #literature #english #computers
Hub usb prodigy.cat PA-01
Review


So, i bought myself this usb hub from allegro. Payed 34,99PLN which is darn cheap. The device connects via usb-c port to the host device and provides usb 3.1 interface with two usb-a ports, two usb-c ports (that support charging the host device), a 4k hdmi port, two micro sd card slots and a gigabit ethernet port. All I tested Is the usb ports and the hdmi port. All work flawless the device is constantly stable as f*ck and I used it quite a time. It has a blue power led which gives very good touch to the device, shows that designers cared. The ethernet device shows in my macOS as Nanjing Qinheng Microelectronics microchip, the network interface gets created. I mainly used the hub to connect logitech f310 gaming controllers but i've transferred a 7GB file over usb and the hub remained stable and the md5sum matched. The hdmi image and sound are just fine. The device can be connected to my Mac, iPad, iPhone and my mom's Samsung Galaxy phone (which allows to display a fully featured desktop over hdmi). Works flawlessly with all, the device registers in the system and gets enabled. I never tested it with windows however I expect it working just fine. Like i said, the hub itself is flawlessly stable. The implementation of the device is 5/5, the quality is very good. What can I say, so far I can recommend it however I didn’t test it in 100%. It’s a proper no name :D You have to be aware tho i never tested it in terms of security but i'm pretty sure it does not install any software in my systems.
